Sobre o projeto

Esse projeto foi feito para uma avaliação da disciplina de Programação Web 2 (PW 2).

Trata-se de um sistema de cadastro de usuários que realiza as quatro operações básicas de um banco de dados (CRUD).

Para esse projeto eu utilizei o SGBD MySQL e PHP para o back-end. O sistema irá armazenar as senhas criptografadas com o hash sha256 Para compor o front-end eu utilizei o framework Bootstrap e ícones do Font Awesome.
